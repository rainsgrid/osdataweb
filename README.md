# 🌊 Oceanography Data Processing Dashboard

Dashboard berbasis Web (Streamlit) untuk mempermudah pengolahan data oseanografi, mulai dari pembersihan data (*spike removal*) hingga analisis pasang surut.

### 🚀 Fitur Utama
* **Cleaning & Filter:** Hapus *spike* otomatis dan pilihan filter (Averaging, MA, Low-pass).
* **Analisis Pasut:** Ekstraksi komponen harmonik (M2, S2, K1, O1) menggunakan **Utide**.
* **Klasifikasi:** Perhitungan nilai **Formzahl (F)** untuk menentukan tipe pasut.
* **Export:** Hasil olahan bisa langsung diunduh dalam format `.csv`.
